¿Qué es este sitio?

Es una plataforma educativa pensada para alumnos de secundaria que desean aprender programación paso a paso, con teoría clara, ejercicios prácticos, retos, y seguimiento de progreso.

Secciones del sitio

  • Inicio: Presentación general y motivación.
  • Teoría: Conceptos fundamentales de programación.
  • Lenguajes: Introducción a lenguajes como HTML, CSS y JavaScript.
  • Ejercicios: Actividades guiadas con espacio para escribir código.
  • Retos: Problemas de lógica y programación más desafiantes.
  • Progreso: Muestra qué ejercicios y retos has completado.
  • Tutoriales: Mini guías para aprender temas específicos.
  • Perfil: Gestiona tu cuenta y puedes cerrar sesión o eliminarla.

Inicio de sesión

No es obligatorio iniciar sesión para usar el sitio. Pero si lo haces, se guardará tu progreso en tu navegador con localStorage.

¿Dónde se guarda el progreso?

Tu navegador guarda la información localmente. Esto significa que si cambias de navegador o computadora, tu progreso no se transferirá (a menos que se conecte a una base de datos como Firebase).

Consejos de uso

  • Guarda tus avances presionando los botones de "Marcar como completado".
  • Utiliza el apartado de perfil para ver tu nombre y cerrar sesión.
  • Revisa el manual o tutoriales si te pierdes en algún punto.

¿Quién hizo esto?

Este sitio fue creado con cariño para apoyar el aprendizaje de programación en estudiantes de secundaria.
