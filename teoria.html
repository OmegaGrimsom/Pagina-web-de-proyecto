Sección de Teoría

¿Qué es la programación?

La programación es el proceso de escribir instrucciones que una computadora puede entender para realizar tareas.

Variables

Las variables son espacios donde podemos guardar información como números, texto, etc.

let nombre = "Ana";

Condicionales

Las condicionales permiten que un programa tome decisiones basadas en condiciones.

let edad = 16;
if (edad >= 18) {
  console.log("Eres mayor de edad");
} else {
  console.log("Eres menor de edad");
}

Bucles

Los bucles sirven para repetir una acción varias veces. Por ejemplo:

for (let i = 1; i <= 5; i++) {
  console.log("Número: " + i);
}
