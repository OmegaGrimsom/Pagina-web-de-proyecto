🔤 Variables

Una variable es como una caja donde guardas algo para usarlo después.

let nombre = "Ana";

Guarda el texto "Ana" en la variable nombre.

✅ Condicionales

Sirven para que el programa tome decisiones. Por ejemplo:

if (edad >= 18) {
  alert("Eres mayor de edad");
} else {
  alert("Eres menor de edad");
}

🔁 Bucles

Los bucles repiten instrucciones. Este ejemplo muestra números del 1 al 5:

for (let i = 1; i <= 5; i++) {
  console.log("Número " + i);
}

🧠 Funciones

Una función guarda acciones que puedes usar muchas veces:

function saludar() {
  alert("¡Hola!");
}
saludar();

📦 Arreglos

Un arreglo guarda varios datos en una sola variable:

let frutas = ["manzana", "banana", "uva"];
console.log(frutas[1]); // banana

Los arreglos comienzan en la posición 0.

🧱 Objetos

Un objeto guarda datos relacionados en una sola estructura:

let persona = {
  nombre: "Luis",
  edad: 14,
  estudiante: true
};
console.log(persona.nombre);

🖱️ Eventos

Los eventos permiten que el usuario interactúe con la página:

document.getElementById("boton").addEventListener("click", function() {
  alert("¡Diste clic!");
});

Esto muestra un mensaje cuando haces clic en un botón con id boton.

🌐 Manipulación del DOM

El DOM es todo el contenido HTML. Puedes cambiarlo con JavaScript:

document.getElementById("titulo").textContent = "Nuevo título";

Este código cambia el texto de un elemento con id titulo.

🧩 Funciones Flecha

Una forma más corta de escribir funciones:

const saludar = () => {
  console.log("Hola mundo");
};

⚙️ Métodos Útiles

Algunos métodos que puedes usar con arreglos:

let numeros = [1, 2, 3, 4];
numeros.push(5); // Agrega 5 al final
numeros.pop();   // Elimina el último número

🧪 typeof

Sirve para saber qué tipo de dato tiene una variable:

let edad = 12;
console.log(typeof edad); // "number"

💬 Comentarios

Los comentarios no se ejecutan. Son para ti o tus compañeros:

// Esto es un comentario de una línea

/* Esto es un 
   comentario de varias líneas */
