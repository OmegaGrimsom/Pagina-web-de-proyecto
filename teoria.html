🔤 Variables

Una variable es como una caja donde guardamos un valor. Por ejemplo:

let nombre = "Ana";

En este ejemplo, estamos guardando el texto "Ana" en la variable llamada nombre.

✅ Condicionales

Sirven para tomar decisiones en tu código. Por ejemplo:

if (edad >= 18) {
  alert("Eres mayor de edad");
} else {
  alert("Eres menor de edad");
}

🔁 Bucles

Los bucles repiten instrucciones. Por ejemplo, un bucle for:

for (let i = 1; i <= 5; i++) {
  console.log("Repetición " + i);
}

🧠 Funciones

Una función es un bloque de código que se puede usar varias veces:

function saludar() {
  alert("Hola, bienvenido");
}
saludar();
