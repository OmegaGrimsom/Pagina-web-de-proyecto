Conceptos Clave

Algoritmo
Conjunto de pasos ordenados que se siguen para resolver un problema.
Array (Arreglo)
Conjunto de datos almacenados en una sola variable, organizados por índices.
Bucle
Estructura que repite un bloque de código mientras se cumpla una condición.
Compilador
Programa que traduce el código fuente a un lenguaje que entiende la computadora.
Condicional
Sentencia que ejecuta acciones dependiendo de si una condición es verdadera o falsa.
Constante
Valor que no cambia durante la ejecución del programa.
Depuración
Proceso de encontrar y corregir errores en un programa.
Función
Bloque de código que realiza una tarea específica y puede reutilizarse.
IDE
Entorno de Desarrollo Integrado; herramienta que facilita escribir y probar código.
Librería
Conjunto de funciones o recursos ya creados que puedes usar en tus programas.
Objeto
Entidad que combina datos (propiedades) y acciones (métodos) en la programación orientada a objetos.
Pseudocódigo
Forma de escribir algoritmos de manera sencilla, usando lenguaje natural y pasos lógicos.
Variable
Espacio en memoria que guarda un valor, el cual puede cambiar durante el programa.
Operador
Símbolo que realiza una operación sobre valores (ejemplo: +, -, *, /).
Sintaxis
Reglas que indican cómo se debe escribir el código en un lenguaje.
Servidor
Computadora o programa que ofrece recursos o servicios a otros equipos llamados clientes.
Cliente
Dispositivo o programa que solicita información a un servidor.
Framework
Conjunto de herramientas y buenas prácticas que facilitan el desarrollo de software.
Base de Datos
Sistema que organiza y almacena información para ser consultada o modificada.
API
Interfaz que permite a diferentes programas comunicarse entre sí.
