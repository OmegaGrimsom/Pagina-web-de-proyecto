🔤 Variables

Una variable es un espacio donde puedes guardar un valor. Ejemplo en JavaScript:

let nombre = "Juan";

🔁 Bucles

Sirven para repetir acciones. Ejemplo:

for(let i = 0; i < 5; i++) {
  console.log(i);
}

⚙️ Funciones

Una función es un bloque de código reutilizable:

function saludar() {
  console.log("Hola");
}
