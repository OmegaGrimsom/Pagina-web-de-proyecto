Tutoriales de JavaScript

🟡 Tutorial 1: ¿Qué es JavaScript?

JavaScript es un lenguaje de programación que permite dar interactividad a las páginas web. Se ejecuta en el navegador.

🟡 Tutorial 2: Mostrar texto en consola

Usamos console.log() para mostrar mensajes en la consola.


console.log("Hola mundo");

🟡 Tutorial 3: Variables

Las variables almacenan información. Se pueden crear con let o const.


let edad = 15;
const nombre = "Ana";
console.log(nombre + " tiene " + edad + " años.");

🟡 Tutorial 4: Condicionales

Sirven para tomar decisiones en el código.


let nota = 8;
if (nota >= 6) {
  console.log("Aprobado");
} else {
  console.log("Reprobado");
}

🟡 Tutorial 5: Bucles

Permiten repetir acciones varias veces. Ejemplo con for:


for (let i = 0; i < 5; i++) {
  console.log("Repetición número " + i);
}

🟡 Tutorial 6: Arreglos

Un arreglo guarda varios valores juntos.


let frutas = ["Manzana", "Banana", "Uva"];
console.log(frutas[0]); // Manzana

🟡 Tutorial 7: Funciones

Las funciones nos permiten agrupar código para reutilizarlo.


function saludar(nombre) {
  console.log("Hola " + nombre);
}
saludar("Carlos");

🟡 Tutorial 8: Eventos

Con JavaScript podemos reaccionar a cosas que pasan en la página, como hacer clic.


// HTML: 
function saludar() {
  alert("Hola desde un botón");
}

🟡 Tutorial 9: Operadores

Permiten realizar operaciones matemáticas o comparaciones.


let suma = 4 + 2;
let resta = 5 - 3;
let comparacion = 10 > 5;
console.log(suma);
console.log(resta);
console.log(comparacion);

🟡 Tutorial 10: Funciones flecha

Una forma más corta de escribir funciones.


const saludar = nombre => {
  console.log("Hola " + nombre);
};
saludar("Ana");

🟡 Tutorial 11: Manipular el DOM

Podemos cambiar elementos de la página con JavaScript.


// HTML: 
Hola

document.getElementById("mensaje").textContent = "Nuevo mensaje";
